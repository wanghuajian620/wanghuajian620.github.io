rss 订阅

  • rss 是什么

    Really Simple Syndication“简易信息聚合”就是RSS的英文原意，简易信息聚合是一般人很难明白的，我就属于二般的（只理解白话的那类人）。说白了 rss 就是从诸多信息中订阅自己想看的那部分，并且它还会定时自动更新。只不过 rss 是 XML 格式的，作为数据源必须借助一些阅读器来浏览。此外它当前的主要版本是 0.91、1.0、2.0的。

  • 怎么把网页上的 rss 数据源拿下来呢

    思路很简单，只需访问数据源的 url，再根据不同 的 rss 版本（0.91、1.0、2.0）来写不同的数据结构，数据结构的组成则是由数据源的内容组成决定的。访问成功后，引用 "encoding/xml" 包做 XML 的解析即可。推荐相关库 go-rss。

  • 那如何把拿下来的数据存到数据库呢

    首先使用 Docker 创建并跑一个 Mysql 数据库，然后引用 "database/sql" 包连接我们创建的数据库。
