JavaScript中的==和===的比较
2018/03/17 javascript javascript 330

误解

==检查等价性

===检查值和类型两者的等价性

正解

==在允许强制转换的条件下检查值的等价性

===是在不允许强制转换的条件下检查值的等价性

强制转换

强制转换有两种形式：明确的和隐含的

明确的强制转换：是可以在代码中明显地看到一个类型转换到另一个类型
隐含的强制转换： 更像是一些操作不明显的副作用引发类型转换
